## About

A hand-built personal portfolio for Aaron Miller — full-stack & agentic-AI engineer. No framework, no build step: one `index.html` with inline CSS and vanilla JS, deployed straight to GitHub Pages.

## Sections

- **About** — bio, focus areas, and a live tech-stack grid
- **Resume** — work history, education, and downloadable PDF
- **Projects** — featured builds with stack tags and source links
- **Contact** — email and social links

## Design

- **Type:** Outfit (display) + JetBrains Mono (code), via Google Fonts
- **Palette:** void-black backgrounds with cyan/magenta accent gradients
- **Effects:** glassmorphism panels, custom CSS properties, responsive layout
- **Footprint:** single self-contained HTML file — instant load, no dependencies to break

## Local preview

```bash
git clone https://github.com/aaaronmiller/website.git
cd website
# open index.html directly, or serve it:
python3 -m http.server 8000   # then visit http://localhost:8000
```

## Deployment

Served by **GitHub Pages** from the `main` branch. Push to `main` and the live site updates automatically.
